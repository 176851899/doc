# axure

## 下载

目前查看原型图的 axure 版本需要在 axure rp9 以上，所以建议用户下载以下两个版本：  
axure rp9 下载地址: <https://www.axure.com.cn/78629>  
axure rp10 下载地址: <https://www.axure.com>

- windows 系统  
  安装步骤参考: <https://blog.csdn.net/a18870525682/article/details/115768018>
- macOS 系统
